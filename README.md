## Getting started
### Dependencies  
* react-dom
* react-redux
* react-router-dom
* react-scripts
* redux
* semantic-ui-css
* semantic-ui-react

### Setup   
To test this application, fork the repo to your own GitHub account and clone it to your local workspace. </br>
 
Install all of the dependencies:    
```
$ yarn install
```  

Start the React application and run it on your local host:
```
$ yarn start
```
